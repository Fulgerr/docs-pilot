# About Test Cloud

Test Cloud is an Enterprise cloud offering designed for application testers. Test Cloud operates as a dedicated interface of Automation Cloud, specialized in application testing, surfacing UiPath services that help you in building a robust test portfolio. With platform capabilities inherited from Automation Cloud, Test Cloud offers a strong automation and testing ecosystem for managing all your testing operations available in the context of a customizable cloud organization.

For more information on Test Cloud and its capabilities, refer to Test Cloud.
