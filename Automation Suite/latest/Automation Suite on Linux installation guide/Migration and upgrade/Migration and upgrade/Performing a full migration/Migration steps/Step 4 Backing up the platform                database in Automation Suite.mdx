# Step 4: Backing up the platform
                database in Automation Suite

To back up the database of
                                the shared platform capabilities in Automation Suite, take the
                                following steps. Note that the default name of this database is
                                        AutomationSuite_Platform.

Log into the Automation Suite SQL Server using SSMS.Right-click Database, select Tasks, and then
                                                Export Data-tier Application.Select the output path for the BACPAC
                                        file.Finish the process.
