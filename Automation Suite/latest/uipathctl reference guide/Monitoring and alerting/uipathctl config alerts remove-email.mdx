# uipathctl config alerts remove-email

# Description

Remove the existing Alertmanager receiver from the Alertmanager config.

uipathctl config alerts remove-email [flags]

# Examples

# Remove existing receiver from alertmanager config
uipathctl config alerts remove-email --name test

# Options

--ASEA          Boolean flag to denote whether the cluster is ASEA or not
  -h, --help          help for remove-email
      --name string   Name of the email configuration
