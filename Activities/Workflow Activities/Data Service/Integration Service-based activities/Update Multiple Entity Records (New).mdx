# Update Multiple Entity Records (New)

# Description

Creates multiple records in bulk for the selected Data Service entity.



This activity is equivalent to the activity of the same name in the pre 24.10 Data Service activities package.

# Project compatibility

Windows | Cross-platform

# Properties

Data Service connection - Select the Data Service connection you established in Integration Service. Access the dropdown menu to select, add, or manage connections.Insert Records - Provide a collection of records to be updated for the selected Data Service entity.Entity - Select the Data Service entity you want to use with the activity. This property supports entities imported from Data Service, or TypeArgument for ImportedEntities type input.
