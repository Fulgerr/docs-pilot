# About the Form experience

The current Form experience allows you to build complex forms, through which you can collect and process user or application input, and control the way they display using custom user and event triggers.

With the current form capabilities you can:Create forms independently from any activity,
                    using the Form Builder.Create your custom forms using HTML files.Display a form to the user or multiple forms at the same time. Additionally, you can display multiple instances of the same form at the same time.Create triggers based on form events, such as a component changing or losing focus.Continue workflow execution, while a form is still showing.Edit and retrieve form values at runtime.Changing the form layout properties at runtime.Execute forms using custom JavaScript.

# Prerequisites

To use the current Form experience:Install Studio 2023.4.Install Form.Activities 23.4.X.
