# Show Apps Message

# Description

Shows a toast message.

# Usage

PropertyValueDescriptionTitleRequiredThe toast title and first line of text displayed.MessageRequiredThe toast message.TypeRequiredThe message type to show:Info (default)SuccessErrorWarningPositionThe position of the toast on the page:TopBottomAuto-dismiss afterOptionalThe number of seconds to keep the toast open before it disappears. Five by default.
