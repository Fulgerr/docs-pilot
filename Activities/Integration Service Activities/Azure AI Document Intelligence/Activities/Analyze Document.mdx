# Analyze Document

# Description

Asynchronously analyze or extract a document using a model.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Model - Select the model from the drop-down list. This field supports String type input.
* Upload By - Select attachment type - File or URL. This field supports String type input.
* Locale - Locale for text recognition and document analysis. This field supports String type input.
* Pages - List of 1-based page numbers to analyze. This field supports String type input.







* Document analysis features - Enable the selected document analysis features. This field supports String type input.
* String index type - Method to compute string offset and length. This field supports String type input.
* Output content format - Format of the content in analyzed result. This field supports String type input.



* Analyze Document - Automatically generated output variable
* Analysis operation identifier - Automatically generated output variable
