# Search Subscribers

# Description

Searches for list members. This search can be restricted to a specific list or it can
                be used to search across all lists in an account.

# Configuration

* 
* Query - The search query used to filter results, representing a subscriber's email, first name, or last name. This field supports String type input.
* Audience - Enter the name to select the audience/list from the available drop-down list or enter a custom audience/list ID. This field supports String type input.









* Search Subscribers - Automatically generated output variable.
