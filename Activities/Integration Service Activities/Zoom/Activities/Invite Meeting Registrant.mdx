# Invite Meeting Registrant

UiPath.Zoom.IntegrationService.Activities.InviteMeetingRegistrant

# Description

Invites a new meeting registrant to a meeting.

# Project Compatibility

Windows | Cross-platform

# Configuration

Connection - The connection established in Integration Service. Access the drop-down to choose, add or manage connections. This field supports String type input.Meeting - A drop-down containing your scheduled meetings.First name - The registrant’s first name.Last name - The registrant’s last name.Email - The registrant’s email address.

# Advanced

OutputRegistrant Id - The ID of the registrant.
