# Upload Content

# Description

Upload content
            such as image, HTML template, etc.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Name - The name of the content. This field supports String type input.
* Content type - Select the content type from the drop-down list.
* File - Click to use a variable. This field supports IResource type input.
* Content customer key - User-supplied unique identifier for the content. This field supports String type input.
* Category ID - Select the category from the drop-down list or enter the ID of the category associated with the content. This field supports Int64 type input.
