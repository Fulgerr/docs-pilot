# Configuration

Card ID - Type the name of the card to select it from the dropdown list or enter a card ID. This field supports String type input.Name - The name of the attachment. Maximum length: 256. This field supports String type input.URL - A URL to attach to the card. Must start with http:// or https://. This field supports String type input.







* Set cover - Determines whether to use the new attachment as a cover for the card. Default value is False. Boolean value.
* Mime type - The mimeType of the attachment. Maximum length: 256. This field supports String type input.



* Cards attachment - Automatically generated output variable.
