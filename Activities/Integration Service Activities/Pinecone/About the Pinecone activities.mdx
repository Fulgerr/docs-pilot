# About the Pinecone activities

The Pinecone activity package is UiPath Integrations Service's activity pack specific to Pinecone. Pinecone is a vector database that can be used to store and search for vectors, or representations of data that can be understood by AI models.

# AI for UiPath activity package

Once established, the connection to the Pinecone instance can be reused in all subsequent child activities.

To learn more about the Pinecone UiPath activities, see the Activities page for a complete activity list and links to the activity details.
