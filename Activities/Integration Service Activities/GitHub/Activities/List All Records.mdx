# List All Records

UiPath.GitHub.IntegrationService.Activities.ListAllRecords

# Description

Lists all records from GitHub.

# Project Compatibility

Windows | Cross-platform

# Configuration

* Connection - The connection established in Integration Service. Access the drop-down to choose, add or manage connections.
* Select Object - Select the type of objects you want to list from the drop-down menu. Depending on the object you select, additional properties are displayed.
