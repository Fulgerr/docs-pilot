# List All Branches

UiPath.GitHub.IntegrationService.Activities.ListAllBranches

# Description

Lists all the branches for a repository in GitHub.

# Project Compatibility

Windows | Cross-Platform

# Configuration

* Connection - The connection established in Integration Service. Access the drop-down to choose, add or manage connections.
* Repository - The name of the repository. The name is not case sensitive.

# Advanced

Options

* Reference - The exact reference of the branch to filter. For example, to just get the main branch, provide main as the value.

Output

* First List branches Record - The first returned record.
* List of List branches - The list of records.
