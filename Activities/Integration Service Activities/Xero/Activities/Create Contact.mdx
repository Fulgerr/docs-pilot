# Create Contact

# Description

Create a contact in Xero.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Contact name - Name of the contact. This field supports String type input.
* Contact status - Status of the contact. This field supports String type input.
* Email address - Email address of the contact. This field supports String type input.







* Primary person first name - First name of the primary person. This field supports String type input.
* Primary person last name - Last name of the primary person. This field supports String type input.
* Primary person email - Email address of the primary person. This field supports String type input.
* Include primary person in email - Whether to include primary person in the email or not. Boolean value.
* Address line 1 - Line 1 of the contact address. This field supports String type input.
* City of the contact address - City of the contact address. This field supports String type input.
* Postal code - Postal code of the contact address. Select an option from the available dropdown list. This field supports String type input.
* Address type - Type of the contact address. This field supports String type input.
* Phone country code - Country code of the contact’s phone number. Select an option from the available dropdown list. This field supports String type input.
* Phone number - The contact's phone number. This field supports String type input.
* Phone type - Type of the contact’s phone number: Default, DDI, Mobile, or Fax. This field supports String type input.



* Contact - Returned newly created contact. Automatically generated output variable.
* Contact ID - The contact ID. Automatically generated output variable.
