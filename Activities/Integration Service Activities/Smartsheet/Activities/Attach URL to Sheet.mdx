# Attach URL to Sheet

# Description

Attaches a file URL to a sheet.

text="Type the content"

# Configuration

* 
* Sheet ID - The ID of the sheet. Click the icon to browse your Smartsheet files. This field supports String type input.
* URL - The URL of the attachment. This field supports String type input.
* Attachment type - The type of attachment. Select an option from the available drop-down list. Note that Dropbox, Egnyte, and Evernote are not supported by Smartsheet.gov accounts.







* Name - Attachment name. This field supports String type input.
* Description - Attachment description. This field supports String type input.



* Attachment - Automatically generated output variable.
