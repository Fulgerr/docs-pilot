# Create Draft Email Campaign

# Description

Create a draft email campaign.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Title - The internal title for the campaign draft. This field supports String type input.
* Subject - The subject of the campaign. This field supports String type input.
* Content type - The content type of the campaign. This field supports String type input.
* Sender email - Select the sender's email from the available drop-down list. This field supports String type input.
* Sender name - The name of the sender that is displayed to the recipient. This field supports String type input.
* List ID - Type a list name to select one from the drop-down list or enter a custom list ID. This field supports Int64 type input.
* Segment ID - Type a segment name to select one from the drop-down list or enter a custom segment ID. This field supports String type input.



* Data ID - Data ID of the current content. This field supports Int64 type input.
* Reply to - The reply-to email address for the campaign. This field supports String type input.
* Is starred? - Whether the campaign is starred or not. Boolean value.



* Campaign ID - The unique identifier for the campaign draft. Automatically generated output variable.
