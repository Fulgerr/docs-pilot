# Get Marketing List Details

UiPath.SendGrid.IntegrationService.Activities.GetMarketingLists

# Description

Retrieves a specific marketing list from SendGrid.

# Project Compatibility

Windows | Cross-Platform

# Configuration

Connection - The connection established in Integration Service.
                        Access the drop-down to choose, add or manage connections.Contact list ID - The ID of the contact list you want to
                        retrieve.

# Advanced

Options

* Contact sample - Specifies whether a contact sample should be returned or not.

Output

* Response - The returned marketing list, stored in a Lists object.
