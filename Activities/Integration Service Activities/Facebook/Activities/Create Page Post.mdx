# Create Page Post

# Description

Create a text/feed page post on Facebook.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Page - Select the page from the drop-down list or enter the page ID. This field supports String type input.
* Message - Text content to be shared. This field supports String type input.
* Link - Optional link URL to be attached in the post. This field supports String type input.







* Publish status - Indicates if the post is published or in draft state. Boolean value.



* Post ID - ID of the share/post. Automatically generated output variable.
