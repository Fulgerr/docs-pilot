# Make Voice Call

# Description

Make an outgoing voice call to a recipient.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Instance - Select the instance from the drop-down list or pass instance ID. This field supports String type input.
* Contact Flow - Select the contact flow from the drop-down list or pass contact flow ID. This field supports String type input.
* From - Select the phone number associated with the Amazon Connect instance. This field supports String type input.
* To - Phone number of the customer. This field supports String type input.
* Attributes - Comma-separated key-value pairs that can be accessed in flows just like any other contact attributes.







* Name - Name of a voice contact that is shown to an agent in the Contact Control Panel. This field supports String type input.
* Description - Description of the voice contact that is shown to an agent in the Contract Control. This field supports String type input.
* Campaign ID - ID of the campaign associated with the outbound communication. This field supports String type input.
* Queue ID - ID of the queue for the call. If you display a queue, the phone displayed for the caller ID is the phone number specified in the queue. If you do not specify a queue, the queue defined in the flow is used. This field supports String type input.



* Contact ID - Automatically generated output variable.
