# Issue Updated

# Description

Triggers when an existing issue is updated.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Project - The unique key of the project in which the issue should be updated. Use the dropdown list to select a project. This field supports String type input.
* Issue type - The type of issue you wish to create (task, story, bug, epic, etc.). This field supports String type input.
* 





Issues - Automatically generated output variable. Issues ID - Automatically generated output variable.
