# About the Confluence Cloud activities

Confluence is a web-based corporate wiki, Their APIs exposer are the primary way to get and modify data in Confluence Cloud and used to interact with Confluence entities, like pages and blog posts, spaces, users, groups, and more.

To learn more about the Confluence Cloud activities, see the Activities page for a complete activity list and links to the activity details.

# How it works

To enable the outbound automation between UiPath and Confluence Cloud, the activities establish an authenticated connection using the UiPath connector for Confluence Cloud.

Before you build your first project, complete the steps in the Authentication guide from the connector documentation.
