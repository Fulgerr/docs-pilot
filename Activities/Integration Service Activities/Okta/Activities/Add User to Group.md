# Add User to Group

# Description

Adds an user to a group in Okta.

# Project Compatibility

Windows|Cross-Platform

# Configuration

* Connection - The connection established in Integration Service. Access the drop-down to choose, add or manage connections.
* Group ID - The name or ID of the group you want to add the user into.
* User ID - The ID of the user to be added to a group.
* Send email - Indicates if an email should be sent for the activation.
