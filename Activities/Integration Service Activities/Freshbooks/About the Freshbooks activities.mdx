# About the Freshbooks activities

FreshBooks is an online invoicing and accounting platform which efficiently streamlines financial management with key features like invoicing, expense tracking, and time management.



To learn more about the Freshbooks activities, see the Activities page for a complete list and descriptions.

# How it works

To enable the outbound automation between UiPath and Freshbooks, the activities establish an authenticated connection using the UiPath Connector for Freshbooks. Before you build your first project, complete the steps in the Authentication guide from the connector documentation.
