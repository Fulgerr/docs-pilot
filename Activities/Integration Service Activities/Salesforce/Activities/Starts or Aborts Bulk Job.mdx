# Starts or Aborts Bulk Job

# Description

Start or abort a bulk upload or bulk download/query job by providing the job ID.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Job ID - The ID of the job. This field supports String type input.
* Action - Action to perform on the job. When started the job is queued for processing. Options: Start or Abort.
* Job type - Select the job type from the available drop-down list: Bulk Upload or Bulk Download.
