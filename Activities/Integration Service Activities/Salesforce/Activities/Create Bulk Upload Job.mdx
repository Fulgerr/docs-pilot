# Create Bulk Upload Job

# Description

Create records asynchronously in bulk for a Salesforce object by providing a CSV
                file.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Select object - Select any standard or custom object from the available drop-down list.
* CSV file - Click to use a variable. This field supports IResource type input.
* CRUD operation - Type of CRUD operation to be performed. Select an option from the available dropdown list.
* Column delimiter - The column delimiter used in the CSV file. Default is COMMA. Select an option from the available drop-down list.
* Line ending - Line ending used in CSV file, marking the end of a data row. Default is Linefeed (LF). Select an option from the available drop-down list: Linefeed (LF) or Carriage return linefeed (CRLF).









* Job ID - The ID of the bulk job that was created. Automatically generated output variable.
