# Search Workers by Name or ID

# Description

Searches for workers by their names or IDs.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Search string or ID - Searches for workers by name or ID. Supports space-delimited search strings for an OR search. This field supports DateTimeOffset type input. The required ID is not the Workday Identifier (WID). It is the ID associated with the record from Workday REST APIs.
* Include terminated workers - Whether to include terminated workers in the output. Boolean value.









* Workers - Automatically generated output variable.
