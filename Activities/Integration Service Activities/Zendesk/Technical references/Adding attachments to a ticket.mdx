# Adding attachments to a ticket

To upload an attachment to a ticket, you must use the Add Comment
            activity.

First, use an Upload
                File activity to upload a file to Zendesk.

Next, in the Add Comment activity, use the output variable of the Upload
                Attachment activity in the Uploads field. Select the Use variable
            input method to select Upload Attachment > File token.
