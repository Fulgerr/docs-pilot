# About the GoToWebinar activities

GoToWebinar is an online webinar platform. After connecting, engage with your audience through interactive polling, survey. Also, extract data from your list of attendees that helps in qualifying leads for your business.

To learn more about the GoToWebinar activities, see the Activities page for a complete activity list and links to the activity details.

Quickstarts

If you want to learn how to start using your GoToWebinar integration and the activities included in the GoToWebinar activity package, see Quickstarts.

How it works

To enable the outbound automation between UiPath and GoToWebinar, the activities establish an authenticated connection using the UiPath Connector for GoToWebinar.

Before you build your first project, complete the steps in the Authentication guide from the connector documentation.
