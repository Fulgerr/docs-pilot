# Send WhatsApp Message

UiPath.Twilio.IntegrationService.Activities.SendWhatsappMessage

# Description

Sends a message with WhatsApp by using a phone number from Twilio.

# Project Compatibility

Windows | Cross-Platform

# Configuration

Connection - The connection established in Integration Service.
                        Access the drop-down to choose, add or manage connections.To - The receiver's phone number that is supported in WhatsApp.From - The sender's phone number that is supported in WhatsApp and is
                        registered with Twilio.Body - The message that you want to send to the receiver.

# Advanced

Response - The sent message, stored in a
                            Curated_whatsapp_message object.
