# Send Email

# Description

Send an email, with the option to add an attachment in Zoho Mail.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* From - Sender's email address. This field supports String type input.
* To - The email address of recipients. This field supports String type input.
* Subject - The main topic or title of your email. This field supports String type input.
* Body - The email's message body. This field supports String type input.
* Attachment - Select to use a variable. This field supports IResource type input.
* CC - The CC email's address. This field supports String type input.
* BCC - The BCC email's address. This field supports String type input.







* Mail format - Whether the email should be sent in HTML or in plain text. This field supports String type input.



* Message - Automatically generated output variable.
* Message ID - Automatically generated output variable.
