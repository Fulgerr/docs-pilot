# Create a word or PDF document

# Description

Merge the input JSON data with word based templates to create dynamic documents.

# Project Compatibility

Windows|Cross-Platform

# Configuration

* Connection - The connection established in Integration Service. Access the drop-down to choose, add or manage connections.
* File - Uploads a file on which the operation is to be performed.
* Json data - Specfiies the JSON data to merge into the document template.
* Output format - The format in which the target document is to be generated.

# Manage Properties Wizard

                added standard/custom fields are available in the Properties panel (in Studio

Output
