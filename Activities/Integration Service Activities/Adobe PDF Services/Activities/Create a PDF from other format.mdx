# Create a PDF from other format

# Description

This activity uses the Create PDF document from non PDF document - POST API to create a PDF document from Microsoft Office documents (Word, Excel, and PowerPoint) in Adobe PDF Services.

# Configuration

* Connection - The connection established in Integration Service. Access the drop-down to choose, add or manage connections.
* File - The file to upload. This field supports String type input.

# Project Compatibility

Windows|Cross-Platform

Options

* Document language - The language of the original document
