# Translate

# Description

Takes input text and a target language and outputs the translated text in the specified language using a Generative AI model.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Text - The text to be translated. This field supports String type input.
* Language - The language to be translated to. Select a language from the available dropdown list.





* Translated text - The translated text. Where supported, the output will be transliterated into the appropriate script of alphabet.
