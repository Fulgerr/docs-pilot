# Create Folder

# Description

Create a folder inside the selected parent folder.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Parent folder - The parent folder of the folder to be created. This field supports IResource type input. Use the Folder icon to browse your Citrix ShareFile folders.
* Folder name - The name of the folder. This field supports String type input.
* Description - Folder description. This field supports String type input.







* Overwrite existing - Whether to overwrite an existing folder. Boolean value.



* Folder ID - The ID of the new folder. Automatically generated output variable.
* Folder - Automatically generated output variable.
