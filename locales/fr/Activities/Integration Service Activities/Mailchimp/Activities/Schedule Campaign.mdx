# Schedule Campaign

# Description

Schedule a campaign at a later date.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Campaign - Type the name of the campaign to select it from the drop-down list or enter a custom campaign ID. This field supports String type input.
* Schedule at - Select the date and time to schedule the campaign. Campaigns may only be scheduled to send on the quarter-hour (:00, :15, :30, :45). Use the calendar widget to select a date. This field supports DateTimeOffset type input.
