# Configuration

* 
* Name - The name of the list. This field supports String type input.
* Contact company - The company name for the list. This field supports String type input.
* Contact address line 1 - The street address line 1 for the list contact. This field supports String type input.
* Contact city - The city for the list contact. This field supports String type input.
* Contact state - The state for the list contact. This field supports String type input.
* Contact country - A two-character ISO3166 country code. Defaults to US if invalid. This field supports String type input.
* Contact zip - The postal or ZIP code for the list contact. This field supports String type input.
* Permission reminder - A description about the permission reminder regarding how the user joined the audience. For example: "You are receiving this email because you opted in at our website." This field supports String type input.
* Campaign default subject - The default subject line for campaigns sent to this list. This field supports String type input.
* Campaign default from name - The default from name for campaigns sent to this list. This field supports String type input.
* Campaign default from email - The default from email for campaigns sent to this list. This field supports String type input.
* Email type option - If this option is set to True, you receive HTML or plain text email. If it is set to False, you receive HTML emails with a plain text alternative backup. Boolean value.
* Contact address2 - The street address line 2 for the list contact. This field supports String type input.
* Contact phone - The phone number for the list contact. This field supports String type input.









* Audience ID - The ID of the new audience/list. Automatically generated output variable.
* Audience - Automatically generated output variable.
