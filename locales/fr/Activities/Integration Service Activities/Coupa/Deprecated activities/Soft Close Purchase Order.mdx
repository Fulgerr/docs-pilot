# Soft Close Purchase Order

UiPath.Coupa.IntegrationService.Activities.PurchaseOrderClose

# Description

Change the status of a purchase order to Soft Closed.

# Project Compatibility

Windows | Cross-platform

# Configuration

* ConnectionId - The connection established in Integration Service. Access the drop-down to choose, add or manage connections. This field supports String type input.
* Action - The action should always be Close.
* Purchase Order Id - The ID of the purchase order to be soft closed.

# Advanced Options

Output

* Response - The soft closed purchase order response.
