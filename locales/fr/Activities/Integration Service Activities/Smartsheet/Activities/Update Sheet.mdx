# Update Sheet

# Description

Update a sheet in Smartsheet.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Sheet - The ID of the sheet. Click the icon to browse your Smartsheet files. This field supports String type input.
* Name - The updated name of the sheet. This field supports String type input.







* Length of day - The project setting for the length of day. Should be in range of 1 to 24. This field supports Double type input.
* Project settings non working days - Non-working days for a project sheet in YYYY-MM-DD format. This field supports String[] type input.
* Project working days - Working days for the project. Select the days from the drop-down list.
* Critical path enabled - Does the user have show critical path turned on for this sheet. Boolean value.
* Display summer tasks - Does the user have display summary tasks turned on for this sheet. Applies to sheets where calendar view has been configured. Boolean value.



* Sheets - Automatically generated output variable.
