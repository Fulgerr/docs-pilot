# Create Sheet from Template

# Description

Creates a sheet from a template.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Template type - The type of the template from which the sheet is to be created. Select one of the available options from the drop-down list: Public, User created.
* Template - The ID of the template for sheet creation. This field supports String type input.
* Sheet name - The name given to the new sheet being created. This field supports String type input.
* Folder - Optional field. Select a folder to create sheets inside it. Click the icon to browse your Smartsheet folders. This field supports String type input.
* Copy components - Additional parameter to create a sheet from template. This field supports String type input.









* Sheet - Automatically generated output variable.
* Sheet ID - Automatically generated output variable.
