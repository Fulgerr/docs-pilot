# Attach URL to Row

# Description

Attach file URL to a row.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Sheet - The ID of the sheet. Click the icon to browse your Smartsheet files. This field supports String type input.
* Row ID - The ID of the row to which the URL should be added. This field supports String type input.
* URL - The URL of the attachment. This field supports String type input.
* Attachment type - The type of attachment. Dropbox, Egnyte, and Evernote are not supported formats for Smartsheet.gov accounts. This field supports String type input.







* Name - The name of the attachment. This field supports String type input.
* Description - The description of the attachment. This field supports String type input.



* Attachments URL - Automatically generated output variable.
