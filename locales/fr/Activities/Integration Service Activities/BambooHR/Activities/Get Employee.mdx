# Get Employee

# Description

Retrieves the record associated to an employee.

# Configuration

* 
* Employee ID - Select the employee from the drop-down list or pass a custom employee ID. You can also retrieve the ID from the output of an event trigger or the Create Employee activity.







Fields - The fields to be returned for the employee. Provide them
                            in a comma-separated format. Use List all  records > Meta
                                Fields for the official field names, where
                                alias is a field name that contains the exact field
                            names of all the available queryable properties (e.g.,
                                firstName,lastName) that can be used as input
                            here



* Get employee - Automatically generated output variable.
