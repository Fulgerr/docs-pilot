# About the Exchangerates activities

Exchangerates offers up-to-the-minute conversion information for a comprehensive range of 168 global currencies.

# Exchangerates activity package

Once established, the connection to the Exchangerates instance can be reused in all activities in the package.

To learn more about the Exchangerates activities, see the Activities page for a complete activity list and links to the activity details.

# How it works

To enable the outbound automation between UiPath and Exchangerates, the activities establish an authenticated connection using the UiPath Connector for Exchangerates.

Before you build your first project, complete the steps in the Authentication guide from the connector documentation.
