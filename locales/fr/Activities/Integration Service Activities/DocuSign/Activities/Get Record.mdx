# Get Record

UiPath.DocuSign.IntegrationService.Activities.GetRecord

# Description

Retrieves a record in DocuSign.

# Project Compatibility

Windows | Cross-platform

# Configuration

* ConnectionId - The connection established in Integration Service. Access the drop-down to choose, add or manage connections. This field supports String type input.
* Select Object - The object you want to delete.
* <Object>ID - The id of the object selected in the Select Object field.
