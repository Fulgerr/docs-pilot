# About the Pipedrive activities

Pipedrive is a cloud-based customer relationship management tool for sales teams to manage and track contacts, leads and deals.



To learn more about the Pipedrive activities, see the  page for a complete activity list and details.

# How it works

To enable the outbound automation between UiPath and Pipedrive, the activities establish an authenticated connection using the UiPath Connector for Pipedrive. Before you build your first project, complete the steps in the Authentication guide from the connector documentation.
