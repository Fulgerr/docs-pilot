# Create Branch

UiPath.GitHub.IntegrationService.Activities.CreateBranch

# Description

Creates a branch from an existing repository in GitHub.

# Project Compatibility

Windows | Cross-Platform

# Configuration

* Connection - The connection established in Integration Service. Access the drop-down to choose, add or manage connections.
* Repository - The name of the repository you want to create the branch in. The name is not case sensitive.
* Branch - The name of the branch you want to create. The name cannot have wildcard characters (ref/heads/master).

# Advanced

Output

* Branch SHA - The output SHA of the created branch.
* Branch - The name of the branch you want to create. The name cannot have wildcard characters (ref/heads/master)
