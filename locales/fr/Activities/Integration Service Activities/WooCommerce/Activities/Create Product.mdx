# Create Product

# Description

Create a product in WooCommerce, including the price, stock quantity, etc.

# Configuration

Name - The name of the product. This field supports String type input.Type - The type of the product. Select an option from the available dropdown list: Simple, Grouped, External, Variable.Description - Details of the product. This field supports String type input. Short description - Summary of the product. This field supports String type input. Catalog visibility - Catalog visibility of the product. Select an option from the available dropdown list: Visible, Catalog, Search, Hidden. Tax class - Select the tax class of the product: Reduced rate, Standard rate, Zero rate.Shipping class - Select the shipping class of the product: Express, Ground, Priority, Standard.Tags - One or more tags associated with the product. This field supports String type input. Categories - One or more categories associated with the product. This field supports String type input. Featured - Whether the product is featured or not. Boolean value. Stock quantity - Stock quantity of the product. This field supports Int32 type input. Regular price - Regular price of the product. This field supports String type input. Image URLs - One or more comma-separated source image URLs. This field supports String[] type input.





Use Manage Properties to access and set up custom fields, i.e. account specific fields (to capture custom data). These custom fields are available under the {…} attributes section.



* Slug - Slug of the product. This field supports String type input.
* SKU - SKU of the product. This field supports String type input.
* Length - Length of the product. This field supports String type input.
* Width - Width of the product. This field supports String type input.
* Height - Height of the product. This field supports String type input.
* Status - Status of the product. Select an option from the available dropdown list: Draft, Pending, Private, Publish.
* Tax status - Tax status of the product. Select an option from the available dropdown list: Reduced rate, Standard rate, Zero rate.



* Product ID - The ID of the product. Automatically generated output variable.
* Product - Automatically generated output variable.
