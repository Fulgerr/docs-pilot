# About the Marketo activities

Marketo is a cloud-based marketing platform. After connecting, add automation to managing campaigns, SEO work, leads, content, and more. The Marketo activity package enables you to automate Marketo processes. It connects to the service via the UiPath connector for Marketo. You can find out more information about our connector from the About Marketo page.

To learn more about the Marketo activities, see the Activities page for a complete activity list and links to the activity details.

# How it works

To enable the outbound automation between UiPath and Marketo, the activities establish an authenticated connection using the UiPath connector for Marketo.

Before you build your first project, complete the steps in the Authentication guide from the connector documentation.
