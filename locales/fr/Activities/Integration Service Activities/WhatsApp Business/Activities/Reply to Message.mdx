# Reply to Message

# Description

Send a reply to a WhatsApp message or a button click.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Message ID - The ID of the message being replied to for context. This field supports String type input.
* Receiver phone - The destination phone number. This field supports String type input.
* Sender phone ID - Select the sender phone or pass a phone number ID. This field supports String type input.
* Message type - Type of the WhatsApp message. This field supports String type input.

You can get status 200 (successful) almost all the time,
                even when the file is not a valid file. However, message may not get delivered to
                the destination phone number if the file is invalid.





* Message ID - Automatically generated output variable.
