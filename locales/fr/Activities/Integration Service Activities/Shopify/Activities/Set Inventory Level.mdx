# Configuration

Inventory item ID - The ID of the inventory item associated with the inventory level. This field supports String type input.Location ID - The ID of the location that the inventory level belongs to. You can use the dropdown list to find the location. This field supports Int64 type input.Quantity available - Sets the available inventory quantity. This field supports Int64 type input.





* Inventory level - This field supports String type input.
