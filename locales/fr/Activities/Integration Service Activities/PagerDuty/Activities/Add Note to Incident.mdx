# Add Note to Incident

# Description

Adds a note to an incident in PagerDuty.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* From - The email address of a valid user associated with the account making the request. Select an email from the available drop-down list. This field supports String type input.
* Incident ID - The ID of the incident to add notes. Select an ID from the available drop-down list. This field supports String type input.
* Content - The text content of the incident note. This field supports String type input.









* Notes - Automatically generated output variable.
* Note ID - Unique identifier for the note associated with the incident. Automatically generated output variable.
