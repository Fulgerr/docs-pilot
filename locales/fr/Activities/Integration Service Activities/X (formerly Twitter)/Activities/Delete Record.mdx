# Delete Record

UiPath.Twitter.IntegrationService.Activities.DeleteRecord

# Description

Deletes a record on X.

# Project Compatibility

Windows | Cross-platform

# Configuration

* ConnectionId - The connection established in Integration Service. Access the drop-down to choose, add or manage connections. This field supports String type input.
* Select Object - The object you want to configure. Access the drop-down to choose the object.
* id - The user ID.
* Id - The ID of the object selected in the Select Object field. This field changes its name according to what object is selected. For example, choosing Likes as the object changes the name of this field to likesId.
