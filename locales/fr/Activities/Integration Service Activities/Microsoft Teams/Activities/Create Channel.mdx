# Create Channel

# Description

Creates a new channel in a team.

# Untitled Section

# Configuration

* 
* Display name - The display name of the new channel. This field supports String type input.
* Channel type - The type of channel being created. Use Standard for sharing the channel with all team members, Private for sharing the channel with selected team members, and Shared for sharing the channel with people both inside and outside the team.
* Team - The ID of the team the channel belongs to. Enter the name of the team to retrieve it from the available drop-down list or type in a custom team ID. You can retrieve the ID from the output of the Get Team by Name activity. This field supports String type input.
* Description - The description of the new channel. This field supports String type input.









* Channel ID - Automatically generated output variable.
* Teams channels - Automatically generated output variable.
