# Get Content

# Description

Returns a single page of content, like a page or a blog post.

# Project type compatibility

Windows | Cross-platform

# Configuration

* 
* Content type - The type of content to filter by. Options available in the dropdown menu are: page, blog post. This field supports String type input.
* Space key - The key of the space to be queried for its content. A list of spaces is available in the dropdown menu. This field supports String type input.



* Content - The content returned. This field supports only Contents variables.
