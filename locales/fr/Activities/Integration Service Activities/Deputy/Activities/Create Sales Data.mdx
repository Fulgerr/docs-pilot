# Create Sales Data

# Description

Add new sales data for employee.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Area or department -The area or department of the sales. This field supports String type input.
* Transaction date time - The exact date and time of the transaction. Use the calendar widget to select a date. This field supports DateTimeOffset type input.
* Sales reference - The reference number of the sales. This field supports String type input.
* Employee ID - The ID of the employee who made the sale. This field supports String type input.
* Sales type - The type of sales. This field supports String type input.
* Sales amount - The total amount of the sales. This field supports Double type input.









* Sales Data - Automatically generated output variable.
* Sales data ID - Unique identifier for an individual sales data entry. Automatically generated output variable.
