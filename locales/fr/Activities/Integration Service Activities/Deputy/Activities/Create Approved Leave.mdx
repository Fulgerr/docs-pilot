# Create Approved Leave

# Description

Create an approved leave on behalf of employee.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Employee ID - The ID of the employee. This field supports String type input.
* Date start - The start date of the leave. Use the calendar widget to select a date. This field supports DateTimeOffset type input.
* Date end - The end date of the leave. Use the calendar widget to select a date. This field supports DateTimeOffset type input.
* Approval comment - The comment for approving the leave. This field supports String type input.









* Leave - Automatically generated output variable.
* Leave ID - A unique identifier for the approved leave entry. Automatically generated output variable.
