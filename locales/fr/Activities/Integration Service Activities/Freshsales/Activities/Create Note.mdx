# Create Note

# Description

Add note to a contact, account, or deal.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Target type - The type of record the note is associated with. Select an option from the dropdown list: Lead, Contact, Account, or Deal. This field supports String type input.
* Targetable ID - The unique identifier for the record associated with the note. This field supports Int64 type input.
* Note - The detailed text content of the note. This field supports String type input.









* Note ID - The unique identifier for the note. Automatically generated output variable.
* Note - Automatically generated output variable.
