# Update Account

# Description

Update an existing account in Freshsales.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Account ID - The ID of the account. This field supports String type input.
* Name - The name of the account. This field supports String type input.
* Website - The website of the account. This field supports String type input.
* Phone - The phone number of the account. This field supports String type input.
* Sales owner ID - Start typing the email of the user who's been assigned the account to retrieve it from the available dropdown list. This field supports String type input.







* Annual revenue - The annual revenue of the account. This field supports Double type input.
* Number of employees - The number of employees of the account. Select an option from the dropdown list, such as 1-10, 11-50, or 201-500. This field supports Int64 type input.
* Industry type ID - Select the industry type to which the account is associated from the dropdown list. This field supports Int64 type input.
* Business type ID - Select the business type to which the account is associated. This field supports Int64 type input.
* Territory ID - Select the territory of the account from the dropdown list. This field supports Int64 type input.



* Account - Automatically generated output variable.
