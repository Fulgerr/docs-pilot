# Create Campaign

# Description

Create a campaign.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Name - The name of the campaign. This field supports String type input.
* Subject - The subject of the campaign. This field supports String type input.
* Sender name - The name of the sender. This field supports String type input.
* Sender email - A valid email address that has been already verified on MailerLite. This field supports String type input.
* HTML Content - A valid HTML content. The account must be on an advanced plan. This field supports String type input.
* Groups - Select the group from the dropdown list or enter group IDs in comma-separated format. If both groups and segments are provided, only segments are used. This field supports String type input.
* Segments - Select the segment from the dropdown list or enter custom segment IDs in comma-separated format. If both groups and segments are provided, only segments are used. This field supports String type input.









* Campaign ID - The campaign ID. Automatically generated output variable.
* Campaign - Automatically generated output variable.
