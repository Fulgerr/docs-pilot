# Configuration

Template - The ID of the source template. Enter an ID or use the dropdown list
            to find the template. This field supports String type input.Filename - The file name of the document. This field supports
              String type input.Data - The JSON object containing the key-value pair of data to be added to the
            PDF. Example: { "name": "Jane Doe" }. This field supports
              String type input.Metadata - The JSON object containing the key-value pair of metadata to be
            added to the PDF. Example: { "_filename": "my-document.pdf" }. This
            field supports String type input.









* Document ID - Automatically generated output variable.

* Document - Automatically generated output variable.
