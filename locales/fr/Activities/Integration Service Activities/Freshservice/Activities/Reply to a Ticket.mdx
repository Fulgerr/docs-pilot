# Configuration

Ticket ID - The ID of the ticket you want to reply to. This field
                        supports String type input.Reply - The reply message to be sent in the ticket. This field
                        supports String type input.







* From email - The email address from which the reply is sent. By default, the global support email is used. This field supports String type input.
* User ID - The ID of the agent/user who is adding the note. This field supports String type input.
* Attachment - The file attachment to upload. This field supports IResource type input.



* Ticket Reply - Automatically generated output variable.
* Conversation ID - Automatically generated output variable.
