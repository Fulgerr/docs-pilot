# Record Created

# Description

Triggers when a new record is created.

# Project Compatibility

Windows | Cross-Platform

# Configuration

Connection - The
                        connection established in Integration Service. Access the drop-down to
                        choose, add or manage connections.Select Object - Select an object to use with this activity.Filter - Filters the
                        events the trigger reacts to.

# Advanced

Output

* Object ID - The ID of the record that triggered the event.
* Object - The record that triggered the event.
