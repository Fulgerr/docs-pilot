# Upload Receipt Image

UiPath.SapConcur.IntegrationService.Activities.UploadExpenseReceipt

# Description

Uploads a receipt file of up to 10 MB in size (PDF, JPG, JPEG, PNG).

# Project compatibility

Windows | Cross-platform

# Configuration

Connection - The connection established in Integration Service. Access the dropdown menu to choose, add, or manage connections. This field supports strings or String variables.Receipt - The file you wish to upload.
                        This field accepts IResource variables.

Advanced options

OutputFile ID - The ID of the uploaded file. This field supports strings or String variables.
