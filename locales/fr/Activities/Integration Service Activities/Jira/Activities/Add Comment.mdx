# Add Comment

# Description

Add a new comment on an issue in Jira.

# Configuration

Issue ID or key - The issue you wish to add a comment to. Enter
                            the full issue key, such as ABCD-1234, or the issue ID.
                            You can also use the dropdown list to search for an issue. This field
                            supports String type input.Comment - Provide input for the comment using text or HTML. This
                            field supports String type input.









* Issue comments - Automatically generated output variable.
* Comment ID - The ID of the new comment. Automatically generated output variable.
