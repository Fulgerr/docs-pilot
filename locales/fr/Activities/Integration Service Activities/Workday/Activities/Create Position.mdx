# Create Position

# Description

Creates an open position in Workday asynchronously and returns a position event
                ID.

# Untitled Section

# Configuration

* 
* Supervisory organization - The supervisory organization for which this position is being created.
* Job title - The job title of the position being created.
* Job description - The description of the position being created.
* Availability date - Th date when the position becomes available.
* Earliest hire date - The earliest date the applicant can be hired/contracted into this position.
* Is critical - If set to true, creates the position as critical.









* Position event ID - Automatically generated output variable.
* Position event - Automatically generated output variable.
