# Configuration

Board - Select the board from the dropdown list or enter a board ID. This field supports String type input.List - Select the list from the dropdown list or enter a list ID. This field supports String type input.Name - The name for the card. This field supports String type input.Card source - Select the name of the card to select from or enter a card ID. You can use the dropdown list to select a card. This field supports String type input.Keep from source - This field is used along with Card source to specify which properties to copy. Select an option from the available dropdown list. Default value is All.







* Due complete - Whether the due date should be marked as complete. Boolean value.
* Start date - The start date of a card. Use the calendar widget to select a date. Can be left blank. This field supports DateTimeOffset type input.
* URL Source - A URL starting with http:// or https://. This field supports String type input.
* Position - The position of the new card. Valid values are: top, bottom, or a positive float number. This field supports String type input.
* Due date - The due date for the card. Use the calendar widget to select a date. This field supports DateTimeOffset type input.
* Description - The description for the card. This field supports String type input.



* Card ID - The ID of the new card. Automatically generated output variable.
* Cards - Automatically generated output variable.
