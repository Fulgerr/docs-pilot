# Create Contact

# Description

Create contact in a list.

# Untitled Section

# Configuration

Email - Email of the contact. This field supports
                            String type input. List - Select the list from the dropdown or enter a custom list ID.
                        This field supports String type input. Name - Name of the contact. This field supports
                            String type input.



Custom fields, i.e. account specific fields (to capture custom data), are supported
                and can be accessed through the Manage Properties wizard, under customFieldValues.
