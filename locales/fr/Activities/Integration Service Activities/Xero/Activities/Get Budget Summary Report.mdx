# Get Budget Summary Report

# Description

Get a budget summary report.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Timeframe - The period size to compare. This field supports String type input.
* Date - The report date, e.g., 2014-4.30. Use the calendar widget to select a date. This field supports DateTimeOffset type input.
* Period - The number of periods to compare (integer between 1 and 12). This field supports Int32 type input.





Budget summary report - Automatically generated output variable.
