# Check(String, CheckOptions)

Checks or unchecks a box or radio button, identified by its string target.

Check(
    string target, 
    CheckOptions checkOptions)

targetStringIndicate the radio button or check box, from the Object Repository, to
                        select or clear.checkOptionsCheckOptionsCheckType NCheckType - The action to perform:Check - Select a check box or radio button.Uncheck - Clear a check box.Toggle - Switch the state of a check box (select if
                                cleared and clear if selected).AlterIfDisabled Boolean - When true, the API
                        executes the action even if the target element is disabled. The default
                        value is false.
