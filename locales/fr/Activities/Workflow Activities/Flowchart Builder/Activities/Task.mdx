# Task

Use the Task activity as a placeholder for an action or operation to be performed, ranging from human in the loop tasks to fully automated ones.

# Properties

Display nameThe name of the Task activity as you want it to be displayed on the Long Running Workflow canvas.

You can change the type of a Task to:

* Agentic Task
* User Task
* Business Rule Task
* Service Task
* Send Task
* Receive Task
* Subprocess
* Detached Error Handler
