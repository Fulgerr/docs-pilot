# Get Environment Folder

UiPath.Core.Activities.GetEnvironmentFolder

Gets the path to the specified system special folder.

# Properties

Input

* SpecialFolder - The SpecialFolder whose path is to be retrieved.

Output

* FolderPath - The full folder path of the selected SpecialFolder.

Common

* DisplayName - The display name of the activity.

Misc

* Private - If selected, the values of variables and arguments are no longer logged at Verbose level.
