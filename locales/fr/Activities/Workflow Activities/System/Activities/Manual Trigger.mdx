# Manual Trigger

UiPath.Core.Activities.ManualTrigger

# Description

When triggered, this activity outputs job data.

# Project compatibility

Windows - Legacy | Windows | Cross-platform

# Windows, Cross-platform configuration

Advanced options

* JobData - Contains job info such as process name, job key, workflow name, user name, user email, and timestamp.

# Windows - Legacy configuration

Properties panel

Common

* DisplayName - The display name of the activity.

Output

* JobData - Contains job info such as process name, workflow name, user name, user email, and timestamp.

Misc

* Private - If selected, the values of variables and arguments are no longer logged at Verbose level.
