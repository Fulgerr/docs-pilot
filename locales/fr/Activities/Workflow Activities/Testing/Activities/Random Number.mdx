# Random Number

UiPath.Testing.Activites.Test.Data.RandomNumber

# Description

Generates a random integer with a given length.

# Project compatibility

Windows - Legacy | Windows | Cross-platform

# Windows - Legacy configuration

Properties panel

Common

* DisplayName - The display name of the activity.

Input

* Decimal Places - Specifies the maximum number of decimals that can be used for the Number field.
* Max - Maximum number range. The maximum number range is exclusive, meaning that if you set Max to 10, then the actual maximum value returned will be 9. For example, if you want 10 to be the maximum value returned, then set Max to 10.1.
* Min - Minimum number range.

Misc

* Private - If selected, the values of variables and arguments are no longer logged at Verbose level.

Output

* Output - Decimal. Charset: Unicode.
