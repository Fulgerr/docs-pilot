# Submit Action

# Description

Saves and submits the changes made by a user upon completing a form or a certain UI interaction in action app tasks.

# Usage

PropertyValueDescriptionOutcomeRequiredThe outcomes of the user action, defined in the action schema.Properties OverrideOptionalUser-specified action property overrides for output and input/output properties.

# Known limitations

* The Submit Action activity returns default values for all properties that are not overridden.
* Input properties cannot be overridden.
* Renaming a property removes the override for that property and needs to added again using the add button
