# Single Excel Process Scope

System.Activities.Business.SequenceX

Opens or reuses an Excel process, loads the Excel project settings and applies them to
            the associated Excel file. This activity supports working with a single Excel process.
            If you need to work with multiple Excel processes use the Excel Process Scope activity.

Place an Use Excel File activity inside a Single Excel
            Process Scope activity to open Excel documents with certain settings already
            applied.

# Properties

Common

* DisplayName - The display name of the activity.

Misc

* Private - If selected, the values of variables and arguments are no longer logged at Verbose level.
