# Licensing robots for attended automations


## Via Orchestrator

To license a robot for attended automations, first upload and activate a license in Orchestrator, then connect the robot to that Orchestrator instance.To connect a robot to Orchestrator:Configure the Interactive Sign-In in Assistant.In the Assistant, use the Client ID connection type, and provide the client credentials.In the Command Line Interface, use the--connect command, and provide the client credentials.
