# ST-ANA-005 - Check Project.json Exists

Rule ID: ST-ANA-005

Scope: Project

# Description

The rule checks whether the workflow has an associated project.json file.

# Recommendation

All projects developed in Studio must have an associated project.json file. If such a file was not found, try reopening the project with Studio and checking it again.
