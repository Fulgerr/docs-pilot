# Project compatibility

Here you can check the availability of the package and its activities with the different project compatibilities in UiPath Studio.


| Activity package | Windows - Legacy | Windows | Cross-platform |
| --- --- --- ---| Persistence.Activities |  |  |  |

To use the Persistence activity package inside Cross-platform projects, with UiPath Studio version 2022.10 or higher, you need to have the following packages installed:

* Persistence.Activities version 1.4.2 or higher.
* FormActivityLibrary.Activities version 2.0.3 or higher. When using FormActivityLibrary.Activities versions lower than 2.0.2, you need to install .NET5 Desktop Runtime separately.
