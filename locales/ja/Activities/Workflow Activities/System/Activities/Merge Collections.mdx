# Merge Collections

UiPath.Activities.System.Collections.MergeCollections

# Description

Combines the elements of two collections in a new collection.

# Project compatibility

Windows | Cross-platform

# Configuration

* Collection - The first collection to be merged.
* Second Collection - The second collection to be merged.Changing the data type of the collection after selecting it in the Collection or Second Collection fields triggers a validation error. The collection with the new data type must be added again to the activity.

Additional options

Output

* Merged Collection - Reference to the resulting collection for use in other activities.
