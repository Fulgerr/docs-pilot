# Find and Replace

UiPath.Activities.System.Text.FindAndReplace

# Project
                Compatibility

The Find and Replace activity allows you
                to find a given text and replace all its occurrences with another given
                text.

Windows | Cross-Platform

* Source text – The source text to be replaced. Click on the "+" button next to the field to select the type of input.
* Value to find - The text you want to replace.
* Replace with – The replacement text.
* Preview - Displays the potential output of the activity. The preview is available only for static text input

Options

* Match Case - Indicates whether the text replacement is case-sensitive. By default, this is set to False.

Outputs

* Updated Text – The updated text after replacement. Click on the "+" button next to the field to select the type of output.
