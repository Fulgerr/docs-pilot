# Test Manager Connected

Rule ID: TA-DBP-003

Scope: Project

# Description

This rule checks for RPA test cases that are not linked to a test case in Test Manager.

You need to connect your project to Test Manager to make this rule work.

# Recommendation

Make sure that your test cases are properly linked so you can assign them to requirements in Test Manager.

# Configure Rule

1. In the Design ribbon, navigate to Analyze File > Workflow Analyzer Settings.
2. Go to Workflow Analyzer to view the rules.
3. Select Test Manager Connected and configure the default action.
4. Click OK to save changes.

# Reset to Default Value

To reset the rule to the default configuration, right-click it and select Reset to default.
