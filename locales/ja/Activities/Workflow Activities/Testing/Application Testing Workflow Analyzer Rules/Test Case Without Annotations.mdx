# Test Case Without Annotations

Rule ID: TA-DBP-005

Scope: Workflow

# Description

This rule checks for RPA test cases without a single annotation.

# Recommendation

Consider using annotations for your RPA test cases to facilitate team collaboration and communication.

# Configure Rule

1. In the Design ribbon, navigate to Analyze File > Workflow Analyzer Settings.
2. Go to Workflow Analyzer to view the rules.
3. Select Test Case Without Annotations and configure the default action.
4. Click OK to save changes.

# Reset to Default Value

To reset the rule to the default configuration, right-click it and select Reset to default.
