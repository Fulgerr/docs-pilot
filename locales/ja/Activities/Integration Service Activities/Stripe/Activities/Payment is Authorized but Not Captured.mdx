# Payment is Authorized but Not Captured

# Description

Triggers when a credit card is charged and authorized.

# Project Compatibility

Windows|Cross-Platform

# Configuration

* Connection - The connection established in Integration Service. Access the drop-down to choose, add or manage connections.
* Filter - Filters the results of the activity.

# Advanced

Output

* Charges ID - The ID of the charges.
* Charges - The charges.
