# Link Opportunity to Contact

# Description

Creates a link for an opportunity in the Insightly CRM.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Opportunity ID - The ID of the opportunity. This field supports String type input.
* Link object name - Select the link object name from the available drop-down list. This field supports String type input.
* Link contact ID - Select the contact ID from the available drop-down list. This field supports String type input.







* Details - Enter details of the link. This field supports String type input.
* Link ID - Select the ID of the link from the available drop-down list. This field supports String type input.
* Relationship ID - Specify the relationship ID. This field supports String type input.
* Role - Specify the role or title. This field supports String type input.



* Link an opportunity to contact - Automatically generated output variable.
