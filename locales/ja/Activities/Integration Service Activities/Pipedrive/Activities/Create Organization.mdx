# Create Organization

# Description

Create an organization in Pipedrive.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Name - The name of the organization. This field supports String type input.
* Visible to - Select the visibility from the drop-down list: Owner & followers / Owner only, Entire company / Owner's visibility group, Owner's visibility group and sub-groups, Entire company.





Custom fields i.e. account specific fields (to capture custom data) are supported for
                this activity.





* Organization - Automatically generated output variable.
* Organization ID - The new organization's ID. Automatically generated output variable.
