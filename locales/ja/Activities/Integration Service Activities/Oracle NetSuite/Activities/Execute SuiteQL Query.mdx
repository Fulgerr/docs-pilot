# Execute SuiteQL Query

# Description

Execute a suiteQL query on NetSuite databases. Only SELECT queries
                are supported.

For details, see .

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Select query - Input a valid SELECT query, then click the Generate output schema button. Query example: SELECT TOP 1 * FROM [TABLE_NAME]. Add a valid query (without variables) to generate the schema. You can add or update variables in the query after the schema is successfully generated.







* Limit - The number of records to return. This field supports Int32 type input.
* Offset - The number of records to skip. This field supports Int32 type input.
