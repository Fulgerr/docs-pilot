# Upload Attachment to Thread

# Description

Upload a file to a thread for later use when sending messages within the same
                thread.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Thread ID - The ID of the thread where the attachment will be used. This field supports String type input.
* Attachment - Click to use a variable. This field supports IResource type input.









Attachment ID - The attachment ID. Automatically generated output
                        variable.Attachment - Automatically generated output variable.
