# OCR

# Description

Detects the text in an image and extracts the recognized characters into a
                machine-usable character stream. Supported image formats: JPEG, PNG, GIF, BMP. Image
                size should be less than 4 MB.

# Configuration

* 

* Language - Select the language in which the output should retrieve the description of the image. The default language is English.
* Orientation - Whether to detect the text orientation in the image or not. Boolean value.







* Image URL - The public URL of an image. This field supports String type input.
* Model-version - This optional parameter specifies the version of the AI model. The default value is latest.



* Full Text - The fullText response of an OCR request. Automatically generated output variable.
* Language - The language response of an OCR request. Automatically generated output variable.
* ocr - Automatically generated output variable.
