# Activate Email Campaign

# Description

Activates an existing campaign in Drip.

# Project Compatibility

Windows|Cross-Platform

# Configuration

* Connection - The connection established in Integration Service. Access the drop-down to choose, add or manage connections.
* Account - Type the name to select the account from the drop-down or pass a custom account ID. Once this field is populated, the Campaign property field becomes available.
* Campaign - Type the name to select the campaign from the drop-down menu or pass a custom campaign ID.
