# Search Products

# Description

Search products based on their title, handle, status, and more.

# Untitled Section

# Configuration

Title - Filter products by product title. This field supports
              String type input.Handle - Filter products by a comma-separated list of product handles. This
            field supports String type input.







* Status - Filter products by their status. This field supports String type input.
* ID - Filter products by a comma-separated list of product IDs. This field supports String type input.



* Products - Automatically generated output variable.
