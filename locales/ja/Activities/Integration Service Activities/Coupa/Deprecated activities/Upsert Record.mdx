# Upsert Record

UiPath.Coupa.IntegrationService.Activities.UpsertRecord

# Description

Upserts a record in Coupa.

# Project Compatibility

Windows | Cross-platform

# Configuration

* ConnectionId - The connection established in Integration Service. Access the drop-down to choose, add or manage connections. This field supports String type input.
* Select Object - The object you want to configure. Access the drop-down to choose the object.
* <Object>Id - The ID of the object selected in the Select Object field. This field changes its name according to what object is selected. For example, choosing Lookup Values as the object changes the name of this field to lookup_valuesId.

# Manage Properties Wizard

You can further configure additional fields for the activity by using the Manage Properties Wizard.

# Advanced Options

Output

* ID - The ID of the record that has been upserted.
