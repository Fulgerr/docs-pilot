# Cancel Purchase Order

UiPath.Coupa.IntegrationService.Activities.PurchaseOrderCancel

# Description

Change the status of a purchase order to Cancelled.

# Project Compatibility

Windows | Cross-platform

# Configuration

* ConnectionId - The connection established in Integration Service. Access the drop-down to choose, add or manage connections. This field supports String type input.
* Action - The action should always be Cancel.
* Purchase Order Id - The ID of the purchase order to be cancelled.

# Advanced Options

* Response - The cancelled purchase order response.
