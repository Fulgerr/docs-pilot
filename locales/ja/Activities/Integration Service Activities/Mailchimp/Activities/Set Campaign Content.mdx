# Set Campaign Content

# Description

Set the campaign content in Mailchimp.

# Configuration

* 
* Campaign - Type the name of the campaign to select it from the drop-down list or enter a custom campaign ID. This field supports String type input.
* URL - The URL where the HTML lives. You must provide either one of the URL, Plain Text, or HTML fields. This field supports String type input.
* Plain text - The plain-text portion of the campaign. If left unspecified, this is generated automatically. You must provide either one of the URL, Plain Text, or HTML fields. This field supports String type input.
* HTML - The raw HTML for the campaign. You must provide either one of the URL, Plain Text, or HTML fields. This field supports String type input.









Campaigns Content - Automatically generated output variable.
