# Get Enriched Contact

# Description

Enrich contact or lead information by supplying details such as email, first name, last
        name, and more.

# Configuration

Output includes - Select the fields to be included as part of the output. Use
              the dropdown list to make your selection.Email address - The email address of the contact. Only one of the following
              fields is required: email address, hashed email address, phone number, or person ID.
              This field supports String type input.Hashed email address - The hashed email address of the contact. Only one of
              the following fields is required: email address, hashed email address, phone number,
              or person ID. This field supports String type input.Phone number - The phone number of the contact. Only one of the following
              fields is required: email address, hashed email address, phone number, or person ID.
              This field supports String type input.Person ID - The ZoomInfo unique person ID. Only one of the following fields is
              required: email address, hashed email address, phone number, or person ID. This field
              supports Int64 type input.First name - The first name of the contact or lead. This field supports
                String type input.Last name - The last name of the contact or lead. This field supports
                String type input.Company name - The name of the company. When effectively combined with contact
              properties, results in a unique individual. This field supports
                String type input.Company ID - The distinctive ZoomInfo company identifier. When effectively
              combined with contact properties, results in a unique individual. This field supports
                Int64 type input.







* Job title - The current employment job title. This field supports String type input.
* External URL - Social media URL, such as LinkedIn, Facebook, Twitter. This field supports String type input.
* Valid date after - The date after which the contact's profile was last validated. Use the calendar widget to select a date. This field supports DateTimeOffset type input.
* Last updated date after - The date after which the contact's profile was last updated. Use the calendar widget to select a date. This field supports DateTimeOffset type input.
* Minimum accuracy score - Returns contacts that is equal or above the provided accuracy score. This field supports Int32 type input.



* Person - Automatically generated output variable.
* Person ID - The ID. Automatically generated output variable.
