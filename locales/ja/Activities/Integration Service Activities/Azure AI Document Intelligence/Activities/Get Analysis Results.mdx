# Get Analysis Results

# Description

Retrieve the analysed or extracted results by providing the operation ID from the
                output of Analyze Document. The results will be retrieved only when the
                output Document processing status of Get Status activity is
                succeeded.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Model - Select the model from the drop-down list. This field supports String type input.
* Operation ID - Pass the operation ID from the output of Analyze Document. This field supports String type input.





* Analyze Results - Automatically generated output variable.
