# Export a PDF to other formats

# Description

Exports a PDF file into a number of supported formats by performing an OCR operation.
                The supported formats are DOC, DOCX, PPTX, XLSX, RTF.

# Project Compatibility

Windows|Cross-Platform

# Configuration

* Connection - The connection established in Integration Service. Access the drop-down to choose, add or manage connections.
* File - Uploads a file on which the operation is to be performed.
* Output format - The target format in which the PDF file is to be exported. The supported formats are DOC, DOCX, PPTX, XLSX, RTF.
* OCR Language - Specifies the OCR language to be used while exporting the PDF.
