# Get Channel by Name

# Description

Retrieves the channel in a team based on the channel name.

# Untitled Section

# Configuration

Team  - The ID of the
                        team the channel belongs to. Enter the name of the team to retrieve it from
                        the available drop-down list or type in a custom team ID. You can retrieve
                        the ID from the output of the Get Team by Name activity. This field supports
                            String type input. Name - The name of the
                        channel you want to retrieve. Enter a custom name or select it from the
                        drop-down list.









* Channel ID - Automatically generated output variable.
* Channel Type - Automatically generated output variable.
* Channel Object - Automatically generated output variable.
