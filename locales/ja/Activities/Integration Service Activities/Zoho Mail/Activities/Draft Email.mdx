# Draft Email

# Description

Create email as draft or template in Zoho Mail.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Account - Select the account from the list of accounts available in Zoho Mail.
* Mode - Whether the content should be saved as a draft or a template.
* From - The email address of the sender. This field supports String type input.
* To - The email address of the recipient. This field supports String type input.
* Subject - Enter the main topic or title of your email here. This field supports String type input.
* Body - The content that should be saved as a draft or template. This field supports String type input.
* CC - The CC email's address. This field supports String type input.
* BCC - The BCC email's address. This field supports String type input.







* Mail format: Whether the email should be sent in HTML format or in the plain text. This field supports String type input.



* Message - Automatically generated output variable.
* Message ID - Automatically generated output variable.
