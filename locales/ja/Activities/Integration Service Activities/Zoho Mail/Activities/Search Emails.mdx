# Search Emails

# Description

Search for emails based on specified criteria in Zoho Mail.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Account - Select the account from the list of account available in Zoho Mail. This field supports String type input.
* Search by - Search by received date or custom filters. This field supports String type input.







* Fields to be included - The field that needs to be included in the retrieved list. This field supports String type input.
* Limit - The number of emails to be retrieved. This field supports Int32 type input.



* Search result - Automatically generated output variable.
