# About the Zoho Mail activity package

Zoho Mail is a secure, ad-free business email solution that enhances organizational communication. It integrates email with calendars, tasks, notes, and contacts for a streamlined workflow. Focused on privacy and user-friendliness, it's ideal for businesses seeking efficient and confidential communication.



To learn more about the Zoho Mail activities, see the  page for a complete activity list and details.

# How it works

To enable the outbound automation between UiPath and Zoho Mail, the activities establish an authenticated connection using the UiPath Connector for Zoho Mail. Before you build your first project, complete the steps in the Authentication guide from the connector documentation.
