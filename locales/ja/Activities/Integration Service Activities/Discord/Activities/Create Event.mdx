# Create Event

# Description

Create a new scheduled event in a guild.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Server - Select the server/guild from the available drop-down list. This field supports String type input.
* Name - The name of the guild/server event. This field supports String type input.
* Scheduled start time - The scheduled start time of the event. This field supports DateTimeOffset type input.
* Type - Select the entity event type. Select an option from the available drop-down list: External, Voice, Stage instance. Stage instance works only if you have an associated Stage channel.
* Description - Details of the event. This field supports DateTimeOffset type input.
* Guild only - Whether the event is only for the guild or server members. Boolean value.
* Scheduled end time - The scheduled end time of the event. This field supports DateTimeOffset type input.









* Create event - Automatically generated output variable.
* Event ID - The event ID. Automatically generated output variable.
