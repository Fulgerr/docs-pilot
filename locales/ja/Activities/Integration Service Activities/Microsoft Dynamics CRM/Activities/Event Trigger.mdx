# Event Trigger

UiPath.MicrosoftDynamicsCRM.IntegrationService.Activities.CreateTrigger

# Description

Creates a trigger. The trigger becomes visible in Integration Service - Triggers once you publish the process containing this activity.

For a full list of available events, see the Microsoft Dynamics CRM events page.

You can only add triggers to the main workflow in Studio. The trigger must be the first activity in the workflow.

# Project compatibility

Windows | Cross-platform

# Configuration

* Connection - The connection established in Integration Service. Access the drop-down menu to choose, add, or manage connections.
* Event - Choose the type of event you want to set as trigger.
* Record - Choose the type of object you wish to trigger events on.
* Data filters - Click the icon to open the Filter builder widget, where you can create and edit filters for this trigger.
* Output - The EventObjectID.
