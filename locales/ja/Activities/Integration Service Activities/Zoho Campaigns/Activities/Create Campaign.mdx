# Create Campaign

# Description

Create an email campaign.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* From email - Enter a valid sender email address that is registered in Zoho Campaigns. This field supports String type input.
* Name - Name of the campaign. This field supports String type input.
* Subject - Subject of the campaign. This field supports String type input.
* Mailing list - Select the mailing list from the available drop-down list or enter a custom list key. This field supports String type input.
* Topic - Select the topic from the available drop-down list or enter a custom topic ID. This field supports String type input.
* Content URL - URL to the HTML content, such as http://www.example.com. This field supports String type input.









* Campaign - Automatically generated output variable.
