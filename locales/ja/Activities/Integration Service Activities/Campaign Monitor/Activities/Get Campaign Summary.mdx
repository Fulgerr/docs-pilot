# Get Campaign Summary

# Description

Retrieve a campaign's summary from Campaign Monitor.

# Configuration

* 
* Client - Type up to three characters of the name to select the client from the available drop-down list or enter a custom client ID. This field supports String type input.
* Campaign - The ID of the campaign to retrieve from Campaign Monitor. Type up to three characters of the name to select the campaign from the available drop-down list or enter a custom campaign ID. You can also retrieve the ID from the output of the Create Campaign activity (after the campaign is sent) or from the List All Records activity, under the Sent Campaigns object. This field supports String type input.





* Campaign summary - Automatically generated output variable.
