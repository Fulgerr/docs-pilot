# Upload Attachment

# Description

Uses the Upload Files - POST
                API to upload a file to Zendesk and attach it to a ticket comment.

# Configuration

* 
* File name - The name of the file. This field supports String type input.
* Attachment - The file you wish to upload. This field supports IResource type input.









File token - The token of the uploaded file. Automatically generated
                        output variable.Attachment ID - The attachment ID of the uploaded file. Automatically
                        generated output variable.Upload file - Automatically generated output variable.
