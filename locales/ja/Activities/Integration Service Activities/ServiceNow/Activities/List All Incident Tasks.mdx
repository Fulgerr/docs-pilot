# List All Incident Tasks

# Description

Lists all incident tasks.

# Configuration

Connection - The connection established in Integration Service. Access the dropdown menu to choose, add, or manage connections. This field supports String type input.Where - Click to add a filter.Max records - The maximum number of records to return.

# Advanced

Output

* First Incident Task Record - The first record returned.
* List of Incident task - The list of records.
