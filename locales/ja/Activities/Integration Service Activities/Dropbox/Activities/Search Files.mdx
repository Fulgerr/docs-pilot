# Configuration

Query - The search query to return files and folders. Check vendor documentation for help. This field supports String type input.File extension - The file type or extension to search for. You can add multiple extensions. This field supports String type input. File category - The category of the file to search for. These are static and set by Dropbox. Select one of the available options from the drop-down list.







* File status - The status of the file being sought. Select one of the available options from the drop-down list: Viewer, Editor.
* File location - The location of the file being specified. Click the Folder icon to browse your Dropbox folders. This field supports IResource type input.
* Account Owner ID - The owner of the file. This field supports String type input.
* Order by - Order by: Relevance, Last modified.



* List of files - Automatically generated output variable.
