# Delete Issue

UiPath.Jira.IntegrationService.Activities.DeleteIssue

# Description

Deletes an issue in Jira.

Elevated permissions in Jira may be required to use this activity.

# Project compatibility

Windows | Cross-platform

# Configuration

* Connection - The connection established in Integration Service. Access the dropdown menu to choose, add, or manage connections. This field supports strings or String variables.
* Issue - The key of the issue you wish to delete. This field supports String type input.
