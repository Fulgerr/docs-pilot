# Send email

# Description

Sends an email through Amazon SES.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* From - Choose or type a value. This field supports String type input.
* Subject - The email's subject. This field supports String type input.
* Body - The email's message body. This field supports String type input.
* To - The email address of recipients. This field supports String type input.
* CC - The CC email's address. This field supports String type input.
* BCC - The BCC email's address. This field supports String type input.
* Body type - The email format. This field supports String type input.
* Reply-to email address - The reply email addresses. This field supports String type input.







* Body character set - The character set to use for the email body. This field supports String type input.
* Configuration set name - The name of the configuration set to use when you send and email. This field supports String type input.
* Return path - The address that you want bounce and complaint notifications to be sent to. This field supports String type input.
* Subject character set - The character set to use for the email subject. This field supports String type input.



* Message ID - Automatically generated output variable.
