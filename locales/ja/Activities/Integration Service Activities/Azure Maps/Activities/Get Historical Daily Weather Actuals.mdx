# Get Historical Daily Weather Actuals

# Description

Retrieve historical daily weather actuals for a specific location.

# Project Compatibility

Windows|Cross-Platform

# Configuration

Connection - The connection established in Integration Service.
                        Access the drop-down to choose, add or manage connections.Latitude - The latitude of the location.Longitude - The longitude of the location.Start date - The starting date for the historical weather data.End date - The date at which the historical weather data
                        concludes.
