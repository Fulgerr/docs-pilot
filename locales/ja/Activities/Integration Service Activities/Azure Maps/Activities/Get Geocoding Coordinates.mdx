# Get Geocoding Coordinates

# Description

Find location coordinates based on properties of an address.

# Project Compatibility

Windows|Cross-Platform

# Configuration

Connection - The connection established in Integration Service.
                        Access the drop-down to choose, add or manage connections.Address line - The official street line of an address relative to the
                        area, as specified by the locality, postal code, or properties.Locality - The locality portion of an address, such as Seattle.County - The county for the structured address.Country - The country for the structured address.Postal code - The postal code portion of an address.

# Advanced

Output

* Geocoding Response - The geocoding response.
* Latitude - The retrieved latitude.
* Longitude - The retrieved longitude.
