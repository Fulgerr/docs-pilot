# Get Daily Weather Forecast

# Description

Retrieve daily weather forecast data for a specific location.

# Project Compatibility

Windows|Cross-Platform

# Configuration

Connection - The connection established in Integration Service.
                        Access the drop-down to choose, add or manage connections.Latitude - The latitude of the location.Longitude - The longitude of the location.

# Manage Properties Wizard

The Manage Properties Wizard can be used to configure or use any of the object’s
                standard or custom fields by selecting and adding them to the activity canvas. The
                added standard/custom fields are available in the Properties panel (in Studio
                Desktop) or under Additional options (in Studio Web).

# Advanced

Options

* Duration - Duration in days.
* Weather unit type - The weather unit type.

Output

* Forecasts - The retrieved forecast.
* Forecast summary phrase - The forecast summary phrase.
