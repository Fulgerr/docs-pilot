# Detect Object

# Description

Detect a custom
                defined object or objects in an image. You can custom define both the objects to
                search for and rules under which said objects should be detected.

# Project
                compatibility

Windows | Cross-platform

# Configuration

* 
* Entities - Entity name and description to search the image for.
* Additional instruction for detection - Optional instructions for evaluation criteria.
* Image - The image to upload. Supported image file format: PNG (.png), JPEG (.jpeg and .jpg). This field supports IResource type input.





* Object Detection - Automatically generated output variable.
* Entities detected - Array of entities detected.
* Names of detected objects - List of names for all objects that were detected.
