# Reformat

# Description

Detect and change the format of a string of text.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Input - String representation of the content to be reformatted from its original format into a different format. This can also correct malformatted inputs (for example, JSON to JSON). This field supports String type input.
* Input format - The input format. This field is optional and the activity automatically detects the format if not set. Select one of the available options from the dropdown list: String, CSV, JSON, XML, YAML, HTML, Markdown.
* Output format -The output format. Select one of the available options from the dropdown list: String, CSV, JSON, XML, YAML, HTML, Markdown. The output is formatted according to the your preference, but will be available as a string data type.









* Reformatted output - The reformatted output.
