# Configuration

Product IDs - Enter the ID of a product from the output of a Search Products activity or of an event trigger. This field supports Int64 type input.Featured - Whether the product is featured or not. Boolean value. Regular price - Regular price of the product. This field supports String type input. Sale price - Sale price of the product. This field supports String type input. Stock quantity - Stock quantity of the product. This field supports Int32 type input. Stock status - Stock status of the product. Select an option from the available dropdown list: Out of stock, On back order, In stock. On sale from - Start date of sale price, in the site's timezone. Use the calendar widget to select a date. This field supports DatetimeOffset type input. On sale till - End date of sale price, in the site's timezone. Use the calendar widget to select a date. This field supports DatetimeOffset type input. Image URLs - One or more comma-separated source image URLs. This field supports String type input.









* Name - The name of the product. This field supports String type input.
* Status - The status of the product. Select an option from the available dropdown list: Draft, Pending, Private, Publish.
* Catalog visibility - Catalog visibility of the product. Select an option from the available dropdown list: Visible, Catalog, Search, Hidden.
* Tax class - Select the tax class of the product: Reduced rate, Standard rate, Zero rate.
* Shipping class - Select the shipping class of the product: Express, Ground, Priority, Standard.
* Tags - One or more tags associated with the product. This field supports String[] type input.
* Categories - One or more categories associated with the product. This field supports String[] type input.



* Product - Automatically generated output variable.
