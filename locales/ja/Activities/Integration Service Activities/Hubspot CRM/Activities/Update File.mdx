# Configuration

* 
* File ID - Select the file from the drop-down list or enter a custom file ID. You can retrieve the file ID from the output of an event trigger, an Upload File activity, or a List All Records → Files (object) activity. This field supports String type input.
* Access - The type of access for the file. Select an option from the available drop-down list: Public indexable, Public not indexable, Private.
* File - Click to use a variable. This field supports IResource type input.









* File updated - Automatically generated output variable.
