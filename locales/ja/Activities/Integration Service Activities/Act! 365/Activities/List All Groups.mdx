# List All Groups

UiPath.Act365.IntegrationService.Activities.ListAllGroups

# Description

Uses the Groups - GET API to
                list all the groups in Act! 365.

# Project Compatibility

Windows | Cross-Platform

# Configuration

The following properties are located in the body of the activity:

* Max Records - The maximum number of records to return.
* orderBy - Specifies the order of the result set.
* PageSize - The number of resources to return in a given page.
* where - The CEQL search expression, or the where clause, without the WHERE keyword, in a typical SQL query. Filter the groups based on name, created date or description.
* First record - The first record returned.
* List - The list of records.
