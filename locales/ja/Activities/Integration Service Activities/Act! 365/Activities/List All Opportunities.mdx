# List All Opportunities

UiPath.Act365.IntegrationService.Activities.ListAllOpportunities

# Description

Uses the Opportunities - GET
                API to list all the opportunities in Act! 365.

# Project Compatibility

Windows | Cross-Platform

# Configuration

* Connection - The connection established in Integration Service. Access the drop-down to choose, add or manage connections.
* where - Filter the activities based on title, stage or expected close date.

# Advanced

Options

* Order by - The field(s) to order by. For example, you can order by firstname ascending.

Output

* Opportunities - The retrieved opportunities.
