# Clone Program

# Description

Clone an existing program in Marketo.

# Project compatibility

Windows | Cross-platform

# Configuration

* 
* Program - Select the program name from the available dropdown list or enter a custom program ID. This field supports String type input.
* New program name - Enter the name of the new program. This field supports String type input.
* Destination folder - Select the parent/destination folder from the dropdown list or enter a custom folder ID. This field supports String type input.
* Destination folder type - Select the folder type from the dropdown list. This field supports String type input.
* New program description - Enter the description of the new program. This field supports String type input.









* Clone program - Automatically generated output variable.
* New program ID - The unique identifier for the cloned program. Automatically generated output variable.
