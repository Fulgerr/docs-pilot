Uses the Create and Update - POST API to create a new lead in Marketo.

# Configuration

* 
* Email - The email address of the lead. This field supports String type input.
* First name - The first name of the lead. This field supports String type input.
* Last name - The last name of the lead. This field supports String type input.
* Phone number - The phone number of the lead. This field supports String type input.
* Company - The company to which the lead is associated to or belongs to. This field supports String type input.
* Lead source - The source from which the lead was obtained. This field supports String type input.
* Lead score - The score of the lead to be added in Marketo. This field supports Int32 type input.



Custom fields, i.e. account specific fields (to capture custom data), are supported and can be accessed through the Manage Properties wizard.







* Address - The address of the lead. This field supports String type input.
* City - The city of the lead. This field supports String type input.
* State - The state of the lead. This field supports String type input.
* Country - Select the country associated with the lead.
* Postal code - The postal code of the lead. This field supports String type input.



* Lead ID - The ID of the lead. Automatically generated output variable.
* Lead - Automatically generated output variable.
