# Add Leads To A List

# Description

Uses the Add to List - POST API to add the specified lead(s) to an existing list in Marketo.

# Untitled Section

# Configuration

* 
* List ID - The list ID. To retrieve the corresponding ID, you can use the dropdown menu and select an ID from the list or you can narrow it down by starting a search in the text box. This field supports String type input.
* Lead IDs - A comma-separated list of lead IDs that need to be added to a particular list. To retrieve the corresponding ID, you can use the drop-down menu and select an ID from the list or you can narrow it down by starting a search in the text box.









* Added leads - Automatically generated output variable.
