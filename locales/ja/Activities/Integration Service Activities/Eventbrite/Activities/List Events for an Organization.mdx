# List Events for an Organization

# Description

Lists the events for a specified organization.

# Project Compatibility

Windows|Cross-Platform

# Configuration

Connection - The connection established in Integration Service.
                        Access the drop-down to choose, add or manage connections.Organization - The organization ID.
