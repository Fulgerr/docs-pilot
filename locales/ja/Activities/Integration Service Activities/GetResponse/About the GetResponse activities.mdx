# About the GetResponse activities

GetResponse is a cloud-based, user-friendly email marketing tool that helps you to create and grow your email list, capture and nurture leads, send different types of emails, and automate workflows.



To learn more about the GetResponse activities, see the Activities page for a complete list and descriptions.

# How it works

To enable the outbound automation between UiPath and GetResponse, the activities establish an authenticated connection using the UiPath Connector for GetResponse. Before you build your first project, complete the steps in the Authentication guide from the connector documentation.
