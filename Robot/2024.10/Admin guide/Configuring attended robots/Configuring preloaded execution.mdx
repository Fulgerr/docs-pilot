# Configuring preloaded execution

You can manage the behavior of preloaded executors by setting specific environment variables on the robot machine, as follows:

Environment variableValueDescriptionUIPATH_PRE_LOADED_EXECUTORNoneOverwrites the default behavior and starts the preloaded executor only when the first job starts.UIPATH_DISABLE_PRE_LOADED_EXECUTORTrueDisables the preloaded executor.
