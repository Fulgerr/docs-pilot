# ST-DBP-023 - Empty Workflow

Rule ID: ST-DBP-023

Scope: Workflow

# Description

The rule checks whether the project contains empty workflow files, including empty sequences, flowcharts, or state machines.

# Recommendation

Make sure to exclude empty files from the project in order to keep only relevant workflows and to reduce the total size of the project.
